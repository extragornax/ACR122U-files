QuickView Help

 

Brief Description

Quick View is a utility program that checks if you have properly configured your ACS reader.

It also provides relevant ACS reader, driver and System Information.

 

Functionality

This tool supports 2 smart card platforms: PCSC (Personal Computer-Smart Card interface) and ACS Proprietary. To view the list of smart card readers installed on your PC, simply double click on the Smart Card Readers Category (ACS Proprietary Readers or PC/SC Readers). Then you may click on the reader name to check for the reader details.

 

Advanced

This feature displays the ATR Value and the Active Protocol of the Card inserted. To do this, check the Advanced mode and select the reader name. It also enables the user to view Diagnostics Log.  

 

 

 

On the Main Menu:

 

Initialize Starts a session.
Reset Resets a session for new initialization (in case new device was installed or uninstalled).
Quit Terminates the program.

 

  On the Log Menu:

 

View Log Displays the diagnostics log.
Save Log Saves the diagnostics log.
Close Log  Close the diagnostics log window.

 

 

  On the Help Menu:

 

Help

Opens this Help Service.
About

Displays information about the QuickView.
